关于项目


我是买家http://woshimaijia.com 创建于2009年5月，由xinqiyang个人维护。

我是买家，买家分享社区。作为一个UGC网站，使用了现在热门的简洁的交互方式，通过用户分享和站方主动发布的媒介来建立起用户和用户，用户和站点之间的联系。

我是买家，是一个开源项目，访问地址：http://code.google.com/p/woshimaijia/ 提供了1.0.0版本的代码



关于xinqiyang


xinqiyang,混迹与北京，现就职于某中文搜索引擎公司，担任后端开发工程师，主要在LAMP上捣鼓代码，对于产品/开发/运营/运维方面均有所涉猎。我是买家，作为一个完全的例子，验证了从产品到最终实现的整个过程，虽然现在没有运营，但是会将这个项目继续下去。

技术分享


我是买家项目所涉及到的技术信息都发布在 我是买家博客 http://blog.woshimaijia.com 中，喜欢技术的各位订阅博文，欢迎技术交流。
